# reactdotnet
React components within an ASP.NET MVC site.

### Result of ###
```
npm install -g generator-aspnetcore-spa
yo aspnetcore-spa (React+Redux)
```

### To run ###
```
npm install -g webpack
dotnet restore
webpack
dotnet build
dotnet run
```
And browse to [http://localhost:5000](http://localhost:5000)
